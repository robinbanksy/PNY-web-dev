Switch Statements and the Ternary Operator

Switch Statements

We previously saw that we could cover multiple different cases using a conditional statement consisting of an arbitrary number of 'else if' clauses. However, this can be cumbersome and less readable than using a switch statement.

Switch statements are particularly useful when you have a variable that you want to compare against multiple possible values. For example:


switch (variable) {
    case value1:
        // code to be executed if expression === value1
        break;
    case value2:
        // code to be executed if expression === value2
        break;
    // more cases...
    default:
        // code to be executed if expression doesn't match any case
}

And, Or, and Not

So far our boolean expressions have been fairly simple, just checking if a single condition is met. If we want to check multiple conditions, we can use logical operators to form compound expressions.

The and operator (&&) returns true if both operands are true.

The or operator (||) returns true if at least one of the operands is true.

The not operator (!) returns the opposite of the operand's boolean value.

Ternary Operator

The ternary operator is a shorthand way of writing an if-else statement. It takes three operands: a condition, an expression to execute if the condition is true, and an expression to execute if the condition is false. The syntax is as follows:


condition ? expressionIfTrue : expressionIfFalse;

For example:


let canVote = age >= 18 ? "Yes" : "No";
