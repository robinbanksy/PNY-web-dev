	SPECIFICATION
-----------------------------------------------------------------

--- Identification ---

Family Name: Honey Pigeon
Weight: Regular, 400
Width: Normal
Style Name: Regular
PS Font Name: HoneyPigeon
Full Name: Honey Pigeon
Menu Name: Honey Pigeon
FOND Name: Honey Pigeon

--- Character Sets ---

1252 Latin 1 -> MS Windows 1252 Western (ANSI)

--- Kerning ---

The font is fully kerned, including special characters like
Ä, Ö etc. Your software has to support kerning though. There
is an afm-file which includes all the metrics, including the
kerning, in case you want to implement it by yourself.

-----------------------------------------------------------------
	INSTALLING
-----------------------------------------------------------------

Install the ttf-file by double-clicking on it.

-----------------------------------------------------------------
	USAGE
-----------------------------------------------------------------

The font is intended to be used in text bodies of smaller sizes.
