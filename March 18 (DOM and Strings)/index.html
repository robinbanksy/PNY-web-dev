String Manipulation and the DOM

Strings

As covered in previous lessons, strings are a core data type in JavaScript. Essentially, a string is a sequence of characters that form some text.

In some cases, we may want to manipulate the characters within a string; for example, we might want to convert all characters to uppercase or lowercase letters, or extract a substring.

There are a number of ways that we can manipulate strings. JavaScript provides several built-in methods for string manipulation. Additionally, we can use various string properties to access information about a string.

String Methods

Some of the most commonly used string methods include:

  • toUpperCase()
    Converts all characters in a string to uppercase letters. 
    
const str = "Hello, World!";
const upperStr = str.toUpperCase();
console.log(upperStr); // Output: "HELLO, WORLD!"
  • toLowerCase()
    Converts all characters in a string to lowercase letters. 
    
const str = "Hello, World!";
const lowerStr = str.toLowerCase();
console.log(lowerStr); // Output: "hello, world!"
  • substring(start, end)
    Extracts a portion of a string and returns it as a new string. 
    
const str = "Hello, World!";
const subStr = str.substring(0, 5);
const subStr2 = str.substring(7, 12);
const subStr3 = str.substring(7); // If the end index is omitted, it extracts until the end of the string.
console.log(subStr); // Output: "Hello"
console.log(subStr2); // Output: "World"
console.log(subStr3); // Output: "World!"
  • charAt(index)
    Returns the character at a specified index in a string. 
    
const str = "Hello, World!";
const char = str.charAt(0);
console.log(char); // Output: "H"
  • indexOf(substring)
    Returns the index of the first occurrence of a specified value in a string. 
    
const str = "Hello, World!";
const index = str.indexOf("World");
console.log(index); // Output: 7
  • split(delimiter)
    Splits a string into an array of substrings based on a specified delimiter. 
    
const str = "Hello, World!";
const words = str.split(", ");
console.log(words); // Output: ["Hello", "World!"]

The Document Object Model (DOM)

We can finally start learning how we can use JavaScript to interact with our web pages. When we link a JavaScript file to our HTML document, it passes a reference to the document object. Just like the objects we've seen, the document object provides properties and methods which allow us to manipulate it, and by extension, the page itself. This pattern is called the Document Object Model (DOM).

The DOM is a model of the HTML document that it corresponds to. Like the HTML document, the DOM is a hierarchical structure, with nodes representing the document's child elements, and, in turn, the children of those elements. By using functions provided by the DOM, we can access objects modelling specific elements on the page, and access and modify their properties.

For example, we could get a reference to an element, such as a paragraph, and then modify its content or style.

Or, we could get a reference to a text input element and access its value.
